QnA

  • What is DOM & what is the purpose of DOM?

    Answer Document object model is a programming interface, it organizes documents in a way so that programming language can easily access all the data and properties of that document.

  • How to select an element from DOM?

    Answer There are 5 methods of selecting an element. Those are -
    • getElementsByTagName()
    • getElementsByClassName()
    • getElementById()
    • querySelector()
    • querySelectorAll()

  • What is event bubble?

    Answer Event bubbling is a kind of propagation where the event first triggers the last children and then heads back step by step to its parents until it reaches the top most ancestor element

  • What is a callback function and why do we use it?

    Answer When we use a function as a parameter for another function, then its called a callback function. We use callback functions to use functions as parameters.
